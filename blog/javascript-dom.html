Javascript and the DOM

Describe the differences between HTML and CSS using an analogy

A web page is essentially a document. HTML (Hyper Text Markup Language) is the language you use to provide your content, structured in a particular manner, on a web page. CSS (Cascading Style Sheets) is the technology that enables you to format all that content and present it as a visually appealing product. Let's use the analogy of a dessert.

If you only had a HTML document with no styling by CSS, you would know the dessert comprises of 30g vanilla ice cream, 10g chocolate ganache and 15g raspberries, and it would be dished up on a white plate with each item placed one below the other.

Now consider plating your dessert creatively. Perhaps you could use a stoneware blue plate, place 10 dots of ganache to form a circular border and within that arrange the raspberries in a spiral form. Finally, finish off with the dollop of ice cream in the centre of the plate. These details which enhance the beauty of your dessert are what would be specified in your CSS file.

Explain control flow and loops using an example process from everyday life

Say you want to put on your jacket. This activity is a process and has a series of sub-tasks. Control flow is the order in which you are going to perform this activity, and is as follows:

  1. Lift jacket up by the collar with your left hand.
  2. Insert right arm into the right sleeve.
  3. Grasp collar with right hand.
  4. Release left hand from collar.
  5. Insert left arm into left sleeve.
  6. If jacket has a zip, then go to sub-task #7, else if jacket has buttons, then go directly to sub-task #8.
  7. Do the following for zipping up a jacket:
    1. ...
    2. ...
    3. ...
  8. For each button, do the following:
    1. ...
    2. ...
The computer is instructed to perform an activity in a similar manner. It is given a series of instructions, i.e., the code, and it executes each line in the code sequentially, unless it comes across certain coniditions. For instance, if it encounters an if-else condition, such as the case of the jacket having a zip or buttons, it would skip over some code and execute only the required lines.

Another scenario is a loop, i.e., there may be sub-tasks which have to be repeated until some condition is met. In our example above, you are provided a set of steps to fasten a button and these steps must be repeated until every button has been fastened. A for loop has been used in this case.

Describe what the DOM is and an example of how you might interact with it

The DOM stands for Document Object model and is the interface for programming languages to interact and manipulate a web page. Here, we consider we have an HTML document and want to use Javascript as the programming language to change the document's content, structure and style dynamically.

The DOM represents the HTML document as a hierarchy of objects. Therefore, the properties and methods of the objects can be accessed and modified using pre-defined functions called APIs.

For instance, the whole HTML document is represented by an object called document which has a method called getElementsByClassName. If the name of a class is provided as input to this API, it will output all the child elements of the document object which belong to that class. If the class name is "blue", we can call:

var blueElements = document.getElementsByClassName('blue')

Explain the difference between accessing data from arrays and objects

Arrays

The items in an array are indexed and square brackets can be used to access each element using the format arrayName[index]. It must be noted that array indexing starts at 0.

For example, consider an array named studentScores containing the scores received by 5 students for a course.

var studentScores = [80, 76, 54, 68, 86]

The first and third students' scores are 80 and 54, respectively, and can be accessed using indices 0 and 2, as follows:

var firstStudentScore = studentScores[0]

var thirdStudentScore = studentScores[2]

Objects

An object is created with a set of properties in a key-value format. Consider an object called course1 with two properties, courseID and finalScores.

var course1 = {courseID: "ABC2021", finalScores: studentScores}

courseID and finalScores are the keys, and their corresponding values are ABC2021 and the array we created earlier, studentScores, respectively. Using objectName.key, you can retrieve the values associated with those keys.

For example, course1.finalScores will return the array of scores [80, 76, 54, 68, 86] as the output. To directly access the 4th student's score from the course1 object, you can use course1.finalsScores[3] to return the value 68.

Explain what functions are and why they are useful

A function is a set of statements, which takes some input parameters, performs some actions using those parameters, and finally may return an output. Functions are useful when the same sub-task needs to be executed, but with different values for the input parameter.

For instance, let's go back to our dessert made of ice cream, ganache and raspberries. You know the proportion of the ingredients to make 1 serving. If you want to scale up the recipe for multiple servings, you can write a function, which takes as input an object that contains the amount of ingredients for 1 serving and the number of people.
