What does it mean to display inline vs inline blocks?

CSS encloses every HTML element inside a box. This is known as the CSS box model. There are different types of boxes, based on the CSS rules for stacking them up. The ability to change the height and margins around the boxes also varies. Therefore, the display and flow of a webpage depends on the type of the boxes.

By default, CSS has assigned different elements to one of 2 box types - Inline and Block. The CSS style display can be used to change the box type.

In summary,

  • Inline boxes do not force line breaks while block boxes do, which means inline boxes stack side by side, while block boxes stack one below the other.
  • The height, width and margins of block boxes can be modified, but not those of inline boxes.
  • Inline-block boxes combine the above two properties and allow boxes to be stacked horizontally, and the height, width and margins to be modified.
